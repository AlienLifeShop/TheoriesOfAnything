## TIER SYSTEM

| Tier 1 — VERIFIED | Peer-reviewed, primary texts, archaeological consensus |
|---|---|
| **Tier 2 — CREDIBLE** | Academic, debated but supported |
| **Tier 3 — SPECULATIVE** | Possible but unverified, popular-author level |
| **Tier 4 — DUBIOUS** | No credible source, contradicted by evidence |

## KEY RULES

1. NEVER fabricate sources — only cite real, verifiable publications
2. Present ALL sides — mainstream, alternative, skeptical
3. Rate every claim: Tier (1-4) + source confidence [N/5] (weighted bibliography score: journals ×3, academic ×2, other ×1)
4. Always include counter-arguments
5. Debunked claims get `[DEBUNKED]` tag — never deleted
6. Apply interpretive lenses: Mythic, Literal, Symbolic
7. Primary texts > Archaeological evidence > Academic analysis > Popular interpretation > Internet claims

## THESIS STATUS (Updated Mar 8, 2026)

The original thesis — **78.9% of 19 ancient serpent traditions were originally POSITIVE** — holds up after expansion to 1,071 documents across 32 sections.

| Dimension | Status |
|-----------|--------|
| 78.9% positive figure | **Holds** — survey was non-cherry-picked; expansion from 13→19 lowered it honestly |
| Demonization timeline (~2,500 years) | **Strengthened** — W27 Canaanite archaeology provides precise dates (Josiah ~640–609 BCE) |
| Cross-cultural independence | **Strengthened** — W, Y, ZC sections add new civilization cases |
| Evidentiary bar | **Raised** — R14 (Isbell Snake Detection Hypothesis) establishes neurological baseline |
| Oral tradition reliability | **Strengthened** — ZC05 demonstrates 7,000–12,000 year oral accuracy |
| Alternative explanations | **Engaged** — neuroreductionist and archetype explanations fully documented |
| Net thesis integrity | **Moderate-to-strong** — survives best counter-evidence; honest about open questions |

**Strongest new support:** R14's own neuroscience identifies the 78.9% positive finding as *anomalous* — the snake detection system predicts fear, not wisdom-giving. The positive default is the puzzle that requires explanation.

**Strongest new challenge:** Neuroreductionist resolution — arousal from snake imagery is valence-ambiguous; ritual context converts fear to awe. Explains *some* positive associations but not the specific civilizational content (teaching agriculture, astronomy, writing) attributed to serpent beings.
