## TIER SYSTEM

| Tier 1 — VERIFIED | Peer-reviewed, primary texts, archaeological consensus |
|---|---|
| **Tier 2 — CREDIBLE** | Academic, debated but supported |
| **Tier 3 — SPECULATIVE** | Possible but unverified, popular-author level |
| **Tier 4 — DUBIOUS** | No credible source, contradicted by evidence |

## KEY RULES

1. NEVER fabricate sources — only cite real, verifiable publications
2. Present ALL sides — mainstream, alternative, skeptical
3. Rate every claim: Tier (1-4) + source agreement [N/5]
4. Always include counter-arguments
5. Debunked claims get `[DEBUNKED]` tag — never deleted
6. Apply interpretive lenses: Mythic, Literal, Symbolic
7. Primary texts > Archaeological evidence > Academic analysis > Popular interpretation > Internet claims
